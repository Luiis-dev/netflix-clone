## ✨ Funcionalidades

- 🧑‍🤝‍🧑 **Seleção de perfis** — tela inicial com múltiplos perfis de usuário
- 💾 **Persistência com localStorage** — o perfil selecionado é salvo no navegador
- 🎞️ **Catálogo dinâmico** — carrosséis organizados por categoria (Ação, Anime, Kids, Documentários)
- 🎨 **Temas por perfil** — visual adaptado com CSS Custom Properties
- ✏️ **Gerenciar perfis** — modo de edição com ícones interativos

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |

## 📁 Estrutura do projeto

```
NETFLIX_IA/
├── assets/          # Imagens e recursos visuais
├── catalogo/        # Páginas e dados do catálogo
├── index.html       # Página principal (seleção de perfis)
├── index.js         # Lógica principal (ES Modules)
├── script.js        # Scripts auxiliares
└── style.css        # Estilos globais
```
